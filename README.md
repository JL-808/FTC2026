# FTC Team 22689
## Connections
### Control Hub
- Motor Port 0: Front Left Motor
- Motor Port 1: Front Right Motor
- Motor Port 2: Back Left Motor
- Motor Port 3: Back Right Motor
- Servo Port 0: Outtake Servo
- I2C0: (IMU)
- I2C1: Pinpoint Computer
- RS485: Expansion Hub
### Expansion Hub
- Motor Port 0: Outtake Motor (Encoder)
- Motor Port 1: Intake Motor
- Motor Port 2: Lift Left Motor (Encoder)
- Motor Port 3: Lift Right Motor (Encoder)
- RS485: Control Hub
